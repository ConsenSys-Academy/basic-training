Node

Node.js, but often shortened to Node, was built by Ryan Dahl in 2009. It is an “open-source, cross-platform, back-end JavaScript runtime environment that [...] executes JavaScript code outside a web browser.” This means that Javascript, once meant only for websites, could now also be used to program servers and other functions.

Until Node, Javascript had mainly been a “front-end” language, used to build websites that faced users, but not the back-end databases that supply those websites with info. With Node, the impact of Javascript has dramatically expanded.

Node also provides a really powerful package (similar to library or module) manager, npm (Node Package Manager). npm is a command line tool that allows you to install packages and dependencies for Javascript projects. These packages are developed by third-party developers and can be found on npmjs.com. There are over a million packages listed on the site!

yarn

Another common package manager you may see, other than npm, is called yarn and was developed by Facebook, although it's now open-source. Yarn was developed to address security and speed issues with npm ( source).

Overall, they have similar command structures and they both play the same role handling dependencies.

Installing Node

Mac

To install Node and npm on Mac, be sure to install Homebrew. To install Homebrew:

You can either visit the Homebrew website or simply paste this into your terminal:

$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

For more information, see the Installation page from Homebrew

Once Homebrew is installed, install Node and npm by running:

$ brew install node

Linux

To install Node and npm on Ubuntu, run the following commands:

$ sudo apt install nodejs && sudo apt install npm
