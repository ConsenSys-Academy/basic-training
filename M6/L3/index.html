Learning More About Javascript

Below are great places to start learning Javascript. It's very rare that one resource will be able to teach you everything, so check out a number of the resources below and don't feel like you have to stick with any of them. If there are resources you really like that are not here, please let us know by editing this page.

  • Code Academy's free Intro to Javascript course
  • The Odin Project's Javascript Basics
  • Udacity: Javascript Basics
  • The Modern Javascript Tutorial
  • Modern JS Cheatsheet "[A] cheatsheet for JavaScript you will frequently encounter in modern projects...[It's to] help developers with basic knowledge who may struggle to get familiar with modern codebases"
  • If you learned Javascript a while and want to get caught up on developments, read "Modern Javascript: Everything You Missed the Last 10 Years."
  • You Don't Know JS A great series of books, for free online, that walk you through Javascript.
