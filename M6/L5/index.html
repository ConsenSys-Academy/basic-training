Frameworks

What is a Framework?

A Javascript framework is a collection of services that allow for a standard way to build websites encompassing database creation, HTML / CSS templating and session management. (source)

Frameworks arose from the need of dynamic websites, like Facebook and Google. These websites push the limits of HTML with the amount of user interaction and data refreshing they needed to do for a typical session. Javascript frameworks, also sometimes called web frameworks, optimized web performance by building extensive and specialized software in Javascript to be injected into a website and refresh itself based on a user’s actions.

The most popular frameworks are Vue.js, Angular.js (maintained by Google) and React.js (maintained by Facebook) (though React is technically a library, not a framework). Here’s a list of other popular Javascript frameworks.

Why Are They Popular?

Javascript frameworks are popular because they can allow for a developer to quickly develop a sophisticated web application. Be warned about Javascript frameworks, however: While they can provide an initial website very quickly, it can be very difficult to alter a website if you’re not familiar with the framework or not comfortable with Javascript.

Scaffold-eth is a great Javascript framework for Ethereum developers as it’s written using React.js and Hardhat. However, note the above warning about the difficulties of altering React.js application if you’re not comfortable!

  • Create-eth-app, a variation of the very popular create-react-app setup command for React projects
