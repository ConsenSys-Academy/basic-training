VSCode Extensions

One of the reasons we're using VSCode is its extensive library of extensions. These are software add-ons developed by an open community to assist with certain aspects of the developer workflow. You can find VSCode extensions in the VSCode Marketplace. There are thousands of extensions, but not all of them are helpful or appropriate for someone just getting started. Here are the ones we'd recommend you install, broken into categories:

Critical for Beginners

  • Live Share an easy way to share your code with someone for pair programming. You need to have a Github account to register, but you need one for Bootcamp anyway! This is great because, if you're having issues with some code or even install certain libraries, you can connect with a mentor or fellow student extremely quickly and easily.
  • Chronicler A screen recorder for VSCode. This is also extremely helpful for getting help. If you're doing something on VSCode and having issues, but can't connect with anyone realtime, use Chronicler to record what is going wrong in VSCode. It requires some additional software, but it guides you through the download process. There's also the option of recording audio, which would help build context for anyone hoping to help you.

General Development (Intermediate)

  • Live Server which allows you to easily start a server in the directory. It creates a "hot reload" server, which immediately reflects any changes once you save it in the project directory.
  • Gitlens "supercharges the Git capabilities built into Visual Studio Code." Adds significantly more capacity for Git in VSCode.
  • Partial Diff Compare (diff) text selections within a file, across files, or to the clipboard
  • Markdown All in One All you need to write Markdown
  • Github Linker Create links to fragments of code in GitHub

Blockchain Development

  • Solidity Visual Developer Built by ConsenSys!
  • Solidity linter Checks valid Solidity syntax. Fair warning, can be over-opinionated for folks!

VSCode Color Themes

Similar to the terminal color themes, VSCode allows for easy installment of a number of color schemes. Again, the goal is to reduce eyestrain, improve readability, catch syntax errors, and look cool! Here are three popular color themes for VSCode:

  • Dracula
  • Night Owl
  • Oceanic Next

Again, we're just skimming the surface here, for both extensions and themes. Poke around and see what you find!
