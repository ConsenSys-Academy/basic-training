Welcome to Basic Training!

We promise it's much kinder and easier than the movies.

Starting to teach yourself programming can be difficult because there are a lot of small things you learn on the way to bigger things. A basic programming blog post assumes a non-trivial amount of existing knowledge. Sometimes, successfully completing the first step of a tutorial will require hundreds of tiny initial steps beforehand.

It's not just online material, even university courses at elite institutions don't teach fundamental skills. We know this because MIT's Computer Science department has a course called "The Missing Semester of Your CS Education." (we'll cite this often!)

This is not only makes starting to learn programming hard, it can also lead to people to use fundamental tools poorly. If we are taught a task only to accomplish another task, we'll typically only learn the minimum effort that gets us to the second. This can often result in a tutorial spiral where you end up completing X without digesting the fundamentals. We are often taught the command line "good enough" to accomplish the job, but not necessarily best practices. This often results in distaste and ! We can then end up compounding the problem by continuing to use these essential tools in inefficient and sometimes agonizing ways.

We built this mini-course to give a space to learn these fundamental tools in a deliberate, focused way. We saw many people signing up for our Blockchain Frontend Bootcamp, having to learn the fundamentals while also learning frontend development, along with blockchain skills. This course hopes to pre-emptively help with many of the issues we've seen people struggle with over the years.

Even if you're very familiar with the technology in these lessons, we encourage you to flip through them. After all, if you're staring at a code editor for hours, shouldn't it be tricked out in the best way possible? Who knows, what you learn here could make your developer experience not only more efficient, but fun!

A Few Things

Luckily, to get comfortable you don't need to know that many things. It's just that they don't typically get explained together all at once.

In this mini-course, we're going to go through five topics:

  • Unix / Linux Operating Systems
  • Command Line
  • Code Editor
  • Git
  • Javascript / Node / Web Frameworks

We will provide some material in these lessons, but we'll mainly be providing links to other places where you can learn more. If you never left the mini-course, after completing it, you'll be able to find your way around a computer pretty well. However, please follow the links within a lesson to dive deeper, deepening your knowledge, and enhancing your developer experience

Let's get started!
