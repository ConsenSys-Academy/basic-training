Unix / Linux Environment

To be successful in this course you must have access to a Unix or Linux based operating system. MacOS is based on Unix and if you're running Ubuntu, that's a Linux distribution.

If you are on a Windows 10 machine, follow this guide to installing a Linux environment on Windows before starting the course.

What is Unix?

Unix is an operating system developed in the 1970s at the famous AT&T Bell Laboratories in Murray Hill, NJ, USA. It was written to make computer development easier for individuals and teams. It was enormously successful and influenced the development of a whole family of languages classified as "Unix-like." These include the operating systems for Apple computers (MacOS) and Android (Linux).

Unix-like languages provide a common framework for developers to learn and share. As a result, we need every student to have access to either a MacOS system, a Linux system, or a Windows 10 machine (we can install Linux on a Windows 10 machine following this tutorial).

How to Install It

Windows

Until recently, Windows has been a hard operating system to learn software development. However, with Windows 10, users can now have access to Windows Subsystem for Linux (WSL). This has dramatically changed the experience of Windows users in this course, for the better!

Before starting the course, be sure to install WSL on your Windows machine, using this tutorial. When we install VSCode later, be sure to make sure the VSCode software integrates with WSL (follow this tutorial here).

Mac

If you are on a Mac operating system, do not worry about installing Linux. The Mac operating system is "Unix-like" and will do just fine for the course!
