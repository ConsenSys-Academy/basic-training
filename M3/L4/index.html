Terminal Clients

Terminal clients, also called terminal emulators, are ways of making the command line interface easier to work with. They are not necessary and the native terminal clients work great. However, some offer certain features, like autocomplete, as well as color and font customization.

Terminal Emulators for MacOS

  • iTerm2 is a very popular terminal emulator that has features such as split pane windows, autocomplete and hotkeys.
  • Hyper is an Electron-based terminal emulator built using HTML,CSS and JS. You can install extensions from the command line using npm.
  • cool-retro-term is a fun terminal emulator that can look like a number of historic terminals from the 70s, 80s and 90s.

Linux

  • Guake is a terminal emulator for Linux similar to iTerm2 for Mac. It's been around since 2007 and has many open source contributors.
  • Hyper is an Electron-based terminal emulator built using HTML,CSS and JS. You can install extensions from the command line using npm.
  • cool-retro-term is a fun terminal emulator that can look like a number of historic terminals from the 70s, 80s and 90s.

Windows
