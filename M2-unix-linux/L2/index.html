Package Managers

What are they?

The operating system, be it Unix or Linux, is the very basic software needed to run a computer. But we'd like to do more interesting things than just type or use the basic software, right? This is where package managers come in.

Package managers "automate the process of installing, upgrading, configuring and removing computer [software] for a computer's operating system" (source). Installing software involves a host whole of operations, including finding the proper source, making sure the computer can run the software, creating directories for the software to store information, keeping the software up to date, and, if desired, removing the software completely from the computer.

Linux comes with multiples variations which we call distros. One of the most popular one is Ubuntu which we will use throughout the course.

Ubuntu Linux's package manager APT is good and we'll learn about it below, but MacOS' does not have a native package manager installed. We'll go over how to install a third-party package manager for Mac called Homebrew. Please note that other ditros might have different package manager like yum for RedHat & Fedora.

Advanced Package Tool (APT)

Since Linux is open-source, it has a free package manager called APT better recognized from its common command apt-get.

Since it is shipped natively with Linux, you do not have to install it. If you would like to learn more about APT, you can read about it on Ubuntu.com here.

The command line section is next, but a preview here: To run APT on your machine, you typically have to use the sudo command. To learn more about sudo, please see here.

Mac's Package Manager: Homebrew

As mentioned above, MacOS does not come with a native package manager. If you're not comfortable with the command line yet, please go through the next module and install Homebrew when you're comfortable.

Homebrew is a popular, open-source package manager for MacOS and Linux, but it's mainly used on MacOS. Homebrew is written in Ruby and can be installed from the command line. You can either visit the Homebrew website or simply paste this into your terminal: 

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

For more information, see the Installation page from Homebrew
